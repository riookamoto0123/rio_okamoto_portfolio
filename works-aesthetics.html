概要

お店や施設をデザイン・建築する建築会社のコーポレートサイト

目的

会社のイメージを伝え、顧客からの依頼を増やす
求人情報を公開で人員を増やす

ターゲット

建築会社を検討中の企業
入社を検討しているユーザー
年齢:20代～40代
男女比:男性6割、女性4割

構成

トップページのファーストビューに大きく写真を載せることで、会社の建築物のイメージを伝えることを意識しました。トップページで簡潔に会社の概要を伝え、もっと見たいと思ったページに飛べるように、それぞれ下層ページを制作しました。

デザイン

誠実さや安全・安心を表現するためにベースカラーを暗めのグリーンに、活発さを表現するためにアクセントカラーをオレンジに設定しました。フォントは落ち着いた、信頼感のある明朝体を使用しました。

制作期間

企画・デザイン　2週間
コーディング　3週間

ツール

Figma / VS Code

言語

HTML / CSS / jQuery
